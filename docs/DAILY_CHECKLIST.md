# Чек-лист дня — что выгружать из ITIGRIS и банка

> 8 файлов в конце рабочего дня. Папка дня: `daily/YYYY-MM-DD/` на твоём ПК.

## 1. ITIGRIS

| # | Где в UI | Параметры | Файл |
|---|---|---|---|
| 1 | Касса → ДДС → **Учёт всех движений** | Период = день, Категория = *, Тип = Учёт всех движений | `moneyMovementReport.xls` |
| 2 | Касса → ДДС → **Учёт доходов** | Период = день, Категория = **Выручка** | `moneyMovementDetailsReport.xls` |
| 3 | Заказы → **Незавершённые заказы** | На дату = конец дня | `uncompletedOrdersReport.xls` |
| 4 | Касса → **Валовая прибыль** | Период = день, Департамент = **Бардина**, Учитывать дату = **Завершения заказа** | `salesProfit_Бардина.xls` |
| 5 | Касса → **Валовая прибыль** | Период = день, Департамент = **Дружбы**, та же опция | `salesProfit_Дружбы.xls` |

## 2. Банки
